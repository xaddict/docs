Set up Facebook connect

Learn how to set up the Facebook connect for the ZOO comment system. It allows the visitors to sign in to their Facebook account and their data like the profile picture will be inserted into the comment as the avatar.

Configure Facebook connect

The easiest way is to open the ZOO config in one browser window and open the Facebook website in another browser window. This way you can simply copy & paste the needed data for this tutorial. We begin with the ZOO extension.

Click on the app instance in the tab bar of the ZOO, then click on Config in the second tab bar.

Click on Comments on the right side. Here you can see the configuration with the Facebook settings.

Leave this browser window open and go to www.developers.facebook.com/ and log in with your Facebook account data.

On the following screen, click on Set Up New Application in the top right corner

Give your new application a name, in my example name it is ZOO comments. Accept the Facebook terms and click on Create Application.

You must also fill in your Site URL details for the FB Connect to work correctly. You have to enter the URL of the site where you are using the ZOO extension. Type it in the field Site URL. It is important to put the / at the end of the URL. After this, click on Save Changes at the bottom.

Now you see the essential information about the Facebook application.

Copy the Application ID and the Secret from this site, switch to your ZOO installation and paste it into the according fields in the comments configuration and enable Facebook connect with the radio button.

Click on Save in the top right corner to save your settings in the ZOO extension.

Using Facebook connect in your comments

After you have set up Facebook connect the visitors of your site will see a Facebook connect button underneath the comment field.

Pressing the button will show a pop up window, where the visitors should enter their Facebook login details.

After this they will be able to post a comment as Facebook user. You are all set.
