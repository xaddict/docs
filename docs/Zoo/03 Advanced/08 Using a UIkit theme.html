Using a UIkit theme

This tutorial describes how to use the ZOO with one of our UIkit based themes (e.g. Nano 3).

App template

Each of our prebuild ZOO apps comes with a UIkit template. Go to the {doc: create-a-new-app-instance#configuration text: configuration screen} of your app and select the UIkit template.

ZOO Modules

All ZOO modules come with a new theme called Uikit_list. Select it in the {doc: set-up-the-zoo-item-module text: modules settings}. For the ZOO item plugins, you should also select the {doc: assign-elements-to-layout-positions text: UIkit layout}.

Element Separators

There are also a couple of new UIkit specific element separators. All of them are prefixed with UIkit_.
