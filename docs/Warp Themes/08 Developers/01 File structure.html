File structure

Get an overview of how all source files of Warp 7 based themes are organized.

Folder/File Description
/css Contains all CSS files of the theme.
/css/ie8.css Will fix basic issues for the Microsoft Internet Explorer 8.
 
/custom This is the place, where you can store all your {doc: customization text: custom markup and additional files}.
/images Contains all image files of the theme.
 
/js Contains all JavaScript files of the theme.
/js/theme.js Responsible for all JavaScript effects and all JavaScript based functions.
 
/layouts Contains the core files responsible for the theme's layout.
/layouts/theme.php Provides the complete HTML markup for the base theme layout. For more information take a look at {doc: theme-layout text: Theme layout}.
/layouts/theme.config.php Includes layout calculations and assembles the theme's CSS and JS files. For more information take a look at {doc: assets text: JS, CSS and LESS}.
/layouts/widget.php Responsible for displaying all widget variations. For more information take a look at {doc: widgets-2 text: Widgets & Modules}.
 
/less All LESS files of your theme are stored in this folder.
/less/uikit Contains all the UIkit theme LESS files, the frontend framework on which Warp is built.
/less/bootstrap Contains all Bootstrap related LESS files. (Joomla 3 only)
/less/theme.less Defines the theme style and imports the UIkit theme. This file will be compiled into CSS and saved in the /styles/STYLE-NAME/css folder if you hit Compile LESS in the theme settins.
/less/customizer.json Defines which customizer parameters will be displayed in default or advanced mode. It groups UIkit variables, sets what can be defined through the color picker, direct input or a select box and adds fonts.
/less/bootstrap.less Imports all Bootstrap related files. This file will be compiled into CSS and saved in the /styles/STYLE-NAME/css folder if you hit Compile LESS in the theme settins. (Joomla 3 only)
 
/styles Here you can find all available style variations. You don't actually work in this folder. All these files are generated and compiled through the customizer.
/styles/STYLE-NAME/style.less Custom set of LESS variables generated through the customizer.
/styles/STYLE-NAME/css Contains all compiled CSS files of your style. Actually a theme.css file and for Joomla 3 only an additional bootstrap.css file.
 
/warp This folder contains the actual Warp 7 core framework.
 
/changelog.php Gives you information on the version number of your Warp theme and all changes that have so far been made.
/config.default.json Stores all theme related default backend parameters.
/config.json Stores all your saved theme related backend parameters. This file will be generated automatically as soon as you save your settings.
/config.php Registers all LESS files for compilation.
/config.xml This file contains the backend parameters of your theme. For more information, take a look at {doc: config text: Config.xml}.
/templateDetails.xml
theme.xml		 This file contains general information on your theme, like the name, release date, directories and module settings. In Joomla it's templateDetails.xml and in WordPress it's theme.xml.
/warp.php This file loads the Warp Framework.
/favicon.ico Will be displayed in your browser bar. To change the favicon, simply replace that file.
/apple_touch_icon.png Will link to your website, if you save it to the home screen of your mobile device. To change the icon, simply replace that file.
