Update

Learn how to update your Warp theme without losing your customizations.

If a newer version of your theme or Warp is available, you will receive a notification in the theme settings. In that case just visit the Download section and download the latest theme package.

Update Warp

  1. Create a copy of your themes /warp folder. That way, you have a backup in case the update fails.
  2. Download any Warp 7 theme and unzip the package.
  3. Copy the /warp folder from the theme and replace the one in your theme with this one.

NOTE Sometimes a Warp update also requires to update the theme to reflect the changes in Warp accordingly.

Update Theme

The file verification in the theme settings shows all modified and missing files in your theme to help you detect which files you have customized before you start updating. To update your theme do the follow these steps.

  1. First of all make a backup of all your customized theme files, so you can recover all modifications at any time.
  2. Download the latest theme version and extract the package to your Desktop.
  3. Copy all files to /templates/THEME-NAME for Joomla or /wp-content/themes/THEME-NAME for WordPress and replace all files.
  4. Recover your modifications by merging your customized files with the new ones.

IMPORTANT We strongly recommend to use the /custom folder for all modifications and make a backup before updating. That way all your changes will be kept when updating your original theme files by replacing them. Learn more {doc: customization text: here}.
